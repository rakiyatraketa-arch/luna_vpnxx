## 🔑 Personal Access Token (для авторизации):

При пуше GitHub попросит авторизацию:

1. Открой: https://github.com/settings/tokens
2. **Generate new token (classic)**
3. Выбери scope: `repo` (полный доступ к репозиториям)
4. Скопируй токен
5. Используй его вместо пароля при `git push`
