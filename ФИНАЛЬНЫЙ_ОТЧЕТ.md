### 1. Android SDK
Для локальной сборки APK нужен Android SDK. Рекомендуется использовать GitHub Actions.

### 2. Xray binary
Xray binary не включен в git (34 MB). Скачивается автоматически при первом запуске.

Все локации используют один VLESS сервер (Финляндия). Для мультисерверности нужно добавить реальные линки в `server.js`.
