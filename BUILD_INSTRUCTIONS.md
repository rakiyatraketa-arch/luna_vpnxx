## 🔨 Как собрать приложение

### Вариант 1: Android Studio (рекомендуется)

```bash
# 1. Открыть проект
cd C:\Users\slayer\Downloads\vpss\V2rayNG
# Открыть папку V2rayNG в Android Studio

# 2. Синхронизировать Gradle
# File → Sync Project with Gradle Files

# 3. Собрать APK
# Build → Build Bundle(s) / APK(s) → Build APK(s)

# APK будет в:
# V2rayNG\app\build\outputs\apk\debug\app-debug.apk
```

### Вариант 2: Командная строка

```bash
cd C:\Users\slayer\Downloads\vpss\V2rayNG

# Сборка debug версии
gradlew.bat assembleDebug

# Сборка release версии (требуется keystore)
gradlew.bat assembleRelease

# APK будет в:
# app\build\outputs\apk\debug\app-debug.apk
```

## 🧪 Как протестировать

### Тест 1: Проверка режима TUN
```bash
# Установить APK
adb install app-debug.apk

# Запустить приложение
adb shell am start -n com.v2ray.ang/.ui.LunaActivity

# Открыть настройки → должна быть настройка "Режим TUN" (включена)
```

### Тест 2: Проверка сохранения конфигурации
```bash
# 1. Подключиться к VPN через UI
# 2. Закрыть приложение (свайп из недавних)
adb shell am force-stop com.v2ray.ang

# 3. Открыть снова
adb shell am start -n com.v2ray.ang/.ui.LunaActivity

# 4. Проверить что сервер остался выбранным
```

### Тест 3: Проверка очистки конфигураций
```bash
# Проверить логи при запуске
adb logcat | grep "Luna"

# Должно быть сообщение об удалении лишних серверов
# и сохранении только VLESS finlandbox.space
```
